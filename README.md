# Itto-jp.github.io
このWebサイトはGithubで作った最初のサイトになります。(2025年11月25日)<br>
勉強を兼ねているため、記載されているコードにはコメントがたくさんついています。<br>
実を言うと、就活のために急ぎで作ったものです...アピールできるような制作物が無くて...<br>
↓↓↓以下にこのサイトにはどのようなことを載せるか記載します。↓↓↓<br>

- 学習ログ
  何かしらコードを書いて見えるものができたら載せようと思います。<br>
  今の段階では、Unity等のゲームエンジンで簡単な仕様を模倣したコード等、、<br>
  プログラマ志望なのでデザインには今は拘らず、色々なコードを書いて実装できる幅や経験を伸ばせたらいいなと考えています。<br>
- 雑談・他
  基本的にはプログラミングについての学習記録を載せるサイトですが、<br>
  なにか日常であった出来事もたまに載せようと思います。<br>
